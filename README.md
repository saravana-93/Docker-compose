🚀 **Hands-on with Docker Compose!**

Today I completed a practical lab on **Docker Compose** by building and running a multi-container application from scratch.

🔹 What I worked on:

* Built a Dockerized **Node.js** application
* Integrated **MySQL 8** using Docker Compose
* Configured **environment variables** with a `.env` file
* Implemented **named volumes** for persistent database storage
* Practiced Docker Compose lifecycle commands (`up`, `ps`, `logs`, `stop`, `start`, `down`)
* Verified the application by accessing it on **port 3000**

This hands-on exercise helped me understand how Docker Compose simplifies managing multi-container applications and provides a consistent deployment workflow across environments.

Looking forward to the next step: integrating **Nginx**, **GitLab CI/CD**, and deploying the application on **AWS EC2**. 🚀
